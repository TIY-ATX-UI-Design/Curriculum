# Mac Shortcuts

## Key Symbols
- Command ⌘
- Shift ⇧
- Option ⌥
- Control ⌃
- Caps Lock ⇪
- Fn

## Shortcuts
- Copy: Command-c
- Paste: Command-v
- Cut: Command-x
- Undo: Command-z
- Select All: Command-a
- Find: Command-f
- Minimize: Command-m
- New: Command-n
- Open: Command-o
- Save: Command-s
- Close Front Window (or tab): Command-w
- Quit: Command-q
- Spotlight: Command-space
- Switch Apps: Command-tab
- Right Click: Control-click
